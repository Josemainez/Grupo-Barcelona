# Grupo Barcelona - Dashboard

Dashboard estatico en un solo archivo para publicar en Vercel.

## Archivos

- `index.html`: dashboard completo con HTML, CSS, JavaScript y datos embebidos.
- `vercel.json`: configuracion para que Vercel sirva el dashboard como SPA.

## Deploy en Vercel

1. Subir esta carpeta a un repositorio de GitHub.
2. En Vercel, elegir `Add New Project`.
3. Importar el repositorio.
4. Framework preset: `Other`.
5. Build command: dejar vacio.
6. Output directory: dejar vacio.
7. Deploy.
